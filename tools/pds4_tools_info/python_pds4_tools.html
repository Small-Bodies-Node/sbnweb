Python PDS4 Tools¶

Introduction¶

Python PDS4 Tools is pure-Python package that can read and display PDS4 data and meta data. This tool supports all common PDS4 data structures as listed in the Supported Data Structures section. The package expects valid PDS4 labels formatted according to the PDS4 Standard.

An PDS4 reader and visualizer for IDL is separately available.

Contact Lev Nagdimunov with questions or comments regarding this tool, or report issues on GitHub.

Installation¶

Python PDS4 Tools is a pure-Python package available in PyPi and CondaForge.

Install an official release¶

To install the package via pip, run this command:

python -m pip install -U pds4_tools

To install the package via conda, run this command:

conda install -c conda-forge pds4_tools

Install from source¶

You may also install the latest development version from GitHub:

git clone https://github.com/Small-Bodies-Node/pds4_tools.git
cd pds4_tools
python -m pip install .

Download¶

Download the development source code from GitHub.
Download the latest official release: PDS4 tools-1.4.tar.gz. Released on March 16, 2025.

Note: A distributable version of the viewer only, which does not require Python, is available.

Requirements¶

Python 2.7 or 3.5+

pds4_read: NumPy
pds4_viewer: NumPy, matplotlib
